# 🦁 AR-Vocab: Next-Gen AR Learning

Welcome back to your AR-Vocab project! This guide will help you get the app running quickly after you restart your laptop or close your workspace.

---

## 🚀 Quick Start Guide

Follow these steps in order to start your development environment:

### 1️⃣ Start the Metro Bundler
Open a terminal in the project folder and run:
```powershell
npm start
```
*Keep this terminal window open while you are testing the app.*

### 2️⃣ Run on Android
Open a **new** terminal window (leaving the first one running) and execute:
```powershell
npm run android
```
*Make sure your Android device is connected via USB with Debugging enabled, or your Emulator is open.*

---

## 🛠️ Maintenance & Shortcuts

If you run into issues or update assets, use these helpful commands:

| Command | Purpose |
| :--- | :--- |
| `npm run setup:android-assets` | **Sync Assets**: Run this if you add new 3D models or sounds. |
| `npm run clean:metro` | **Clear Cache**: Fixes "glitches" or if the app shows old code. |
| `npm run clean:android` | **Deep Clean**: Completely resets the Android build folder. |

---

## 🌟 Pro Tips
- **Voice Recognition**: Ensure you have a stable internet connection for the best phonetic scoring.
- **AR Models**: Place models on a flat, well-lit surface for the best tracking experience.
- **Mastery**: Remember, stars are only awarded for **100% perfect** pronunciation!
